请帮我用 Python Flask 创建一个带 Markdown 文档功能的 Web 应用，要求如下：

## 核心功能
1. 文档展示：读取 docs/ 目录下的 .md 文件，渲染为 HTML 页面
2. 文档编辑：登录用户可编辑 Markdown 文档内容并保存
3. 用户认证：简单的登录/登出功能（用户名密码即可，暂不需要注册）
4. 文档列表：展示所有文档的目录索引页

## 技术栈
- 后端：Flask + Flask-Login
- Markdown 渲染：mistune
- 前端：Bootstrap 5 + highlight.js（代码高亮）

## 具体要求
### 前提
不要更改当前项目已有的接口功能


### 路由设计
- `/` 或 `/docs` - 文档列表首页，显示 docs/ 下所有 .md 文件的标题和链接
- `/docs/<filename>` - 展示某个 Markdown 文档（将 .md 渲染成 HTML）
- `/docs/<filename>/edit` - 编辑文档（需要登录）
- `/login` - 登录页面
- `/logout` - 登出

### 目录结构
请自行规划，需要存图片资源，首次请提取 requirements.txt 文件


### 功能细节
1. 登录验证：简单硬编码一个用户（admin / password123），使用 Flask-Login
2. 文档编辑：使用 textarea 编辑 Markdown 原文，保存时写回对应的 .md 文件
3. 未登录用户访问编辑页时自动跳转到登录页
4. 代码高亮：使用 highlight.js 自动高亮代码块
5. Markdown 渲染时启用表格、删除线、脚注扩展
6. 文档列表页显示每个文档的修改时间
7. 编辑页底部显示“保存”和“取消”按钮，取消返回查看页
8. 文档新建
9. 目录列表及预览: 左侧文档目录列表，右侧目录预览
10. 目录管理: 支持新建目录及子目录，支持文件位置移动及打标签

### 额外要求
- 使用 Bootstrap 5 实现响应式布局
- 在 doc_view.html 页面上，如果是登录用户，显示“编辑”按钮
- 在导航栏显示当前登录状态和用户名
- docs/ 目录如果不存在则自动创建
- 添加一个示例 docs/example.md 文件，内容包含：标题、列表、代码块、表格

请生成完整可运行的代码。
